Metadata Matching

When saving your own song impressions on DiggerHub, the title/artist/album fields all need to match exactly for it to be considered the same song. An original studio release and the live version are not the same. When matching between friends, an exact match is preferable, but a soft match on title and artist is enough for default ratings or recomendations.
